# totaljs-calculator-api

## Install

```bash
    npm install
```

## Usage

### server

```bash
    npm start
```

### client

Send

````json
    query={"op":"+", "ops":[2, 3]}
```

    5

## Docker

Get the docker container

    docker pull gallegogt/totaljs-calculator-api

Use

    docker run -p 8888:8888 gallegogt/totaljs-calculator-api
